Styling Language

Cascading Style Sheets

Without CSS, the web would be plain black text on a white background. CSS is what gives every webpage its visual identity, layout, and life.

What is CSS?

CSS stands for Cascading Style Sheets and it controls how HTML elements look on screen. It handles colours, fonts, spacing, layout, and even animations.

The "Cascading" part means styles can be applied from multiple sources, and when they conflict, the browser follows a set of rules (specificity and order) to decide which one wins.

Three Ways to Apply CSS

  • Inline — directly on an element using the style="" attribute
  • Internal — inside a <style> tag in the HTML <head>
  • External — a separate .css file linked with <link>

The Box Model

Every HTML element is a rectangular box. The CSS box model describes the layers around content: padding, border, and margin. Understanding this is essential for creating precise layouts.

/* CSS Selector & Rule */ h1 { color: #264de4; font-size: 2rem; margin-bottom: 1rem; } /* Class selector */ .card { background: #fff; border-radius: 12px; padding: 1.5rem; box-shadow: 0 4px 20px rgba(0,0,0,0.1); } /* Flexbox layout */ .container { display: flex; justify-content: space-between; align-items: center; }

Key CSS Properties

  • color / background-color — text and background colours
  • font-size / font-family — typography control
  • display: flex / grid — powerful layout systems
  • transition / animation — smooth visual changes
  • media queries — responsive design for different screens
