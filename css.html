Styling Language

Cascading Style Sheets

CSS transforms plain HTML into visually stunning pages. It controls colours, fonts, layouts, spacing, animations, and responsive behaviour.

What is CSS?

CSS (Cascading Style Sheets) is the language used to style and lay out HTML elements. Without CSS, websites would be plain black text on a white background.

The "Cascading" part means multiple style rules can apply to one element and the browser follows rules of specificity and order to decide which one wins.

Key Concepts at a Glance

📦
Box Model
Every element is a box: content → padding → border → margin. This is fundamental for spacing and layout.
🔀
Flexbox
A one-dimensional layout system. Align items in a row or column, space them out, and centre things with ease.
⬛
CSS Grid
A two-dimensional layout system. Define rows and columns, then place elements precisely within them.
📱
Media Queries
Apply different styles at different screen sizes. The key to making sites look great on phones, tablets, and desktops.
🎬
Transitions & Animations
Smooth visual changes with transition and full keyframe animations with @keyframes.
🎨
CSS Variables
Store reusable values: --primary: #4f9cf9, then use them everywhere with var(--primary).

Selectors

Selectors target which HTML elements to style. You can target by tag (h1), class (.card), ID (#hero), or combinations like .nav a:hover.

/* Element, class, ID selectors */ h1 { color: #4f9cf9; font-size: 2rem; } .card { background: #fff; border-radius: 12px; } #hero { min-height: 100vh; } /* Flexbox layout */ .nav { display: flex; justify-content: space-between; align-items: center; gap: 1rem; } /* Media query — mobile styles */ @media (max-width: 768px) { .nav { flex-direction: column; } } /* CSS Variable + Animation */ :root { --accent: #4f9cf9; } .btn { background: var(--accent); transition: transform 0.2s; } .btn:hover { transform: translateY(-3px); }
