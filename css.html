Cascading Style Sheets

CSS is what turns a functional but ugly page into something people actually want to look at. It controls colour, layout, spacing, animation, and how everything adapts to different devices.

What is CSS?

CSS (Cascading Style Sheets) is the styling language for the web. HTML gives the page structure, CSS gives it personality. Without it, every site would just be unstyled black text on a white background — which technically works, but nobody wants that.

The "cascading" part means multiple rules can target the same element at once. The browser resolves conflicts using specificity — a scoring system where IDs beat classes, classes beat tags, and inline styles beat everything. Understanding this is the key to not going crazy when styles don't apply.

Key Concepts

📦
Box Model
Every HTML element is a box: content sits inside padding, padding is surrounded by a border, and border has margin outside it. Spacing problems are almost always a box model misunderstanding.
🔀
Flexbox
One-dimensional layout. Arrange items in a row or column, control their spacing, align them vertically — all without ancient float tricks. Probably the thing I use most.
⬛
CSS Grid
Two-dimensional layout. You define rows and columns, then place things into specific cells. Better than flexbox for page-level structure.
📱
Media Queries
Apply different styles at different screen widths. The main tool for making layouts work on phones without writing a separate stylesheet.
🎬
Transitions & Animations
transition smooths a property change over time. @keyframes lets you write multi-step animations that run on their own schedule.
🎨
CSS Variables
Define a value once with --name: value then use it anywhere with var(--name). The whole theme system on this site runs on CSS variables.

Selectors & Code

CSS /* element, class, ID — order of specificity */ h1 { color: #4f9cf9; font-size: 2rem; } .card { background: #fff; border-radius: 12px; } #hero { min-height: 100vh; } /* flexbox nav — row layout, space between items, vertically centred */ .nav { display: flex; justify-content: space-between; align-items: center; } /* mobile — stack vertically below 768px */ @media (max-width: 768px) { .nav { flex-direction: column; } } /* CSS variable + hover lift */ :root { --blue: #4f9cf9; } .btn { background: var(--blue); transition: transform 0.2s; } .btn:hover { transform: translateY(-3px); }
