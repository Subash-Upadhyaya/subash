Markup Language

HyperText Markup Language

The skeleton of every webpage. HTML provides structure using elements and tags that tell the browser how to display content.

What is HTML?

HTML stands for HyperText Markup Language and it is the basic building block of any webpage. It gives the page its structure using things called tags.

HTML is not a programming language — it cannot make decisions or do calculations. Instead, it is a markup language that annotates content so the browser knows what to display and how.

Core Concepts

  • Elements are defined by opening and closing tags: <p>...</p>
  • Tags can have attributes: href, src, class
  • HTML5 introduced semantic tags like <header>, <nav>, <article>
  • Every webpage must have <html>, <head> and <body>

Semantic HTML

Semantic elements clearly describe their meaning to both the browser and the developer. Using <article> instead of a generic <div> improves accessibility and SEO.

<!-- Basic HTML structure --> <!DOCTYPE html> <html lang="en"> <head> <title>My Page</title> </head> <body> <h1>Hello World!</h1> <p>A paragraph of text.</p> <a href="#">A link</a> </body> </html>
