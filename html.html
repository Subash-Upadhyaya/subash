HyperText Markup Language

The skeleton of every webpage. HTML provides structure using elements and tags — it is the starting point for everything you see on the web.

What is HTML?

HTML (HyperText Markup Language) is the standard language used to create the structure of webpages. It tells the browser what to display such as headings, paragraphs, images, links, forms, and more.

HTML is not a programming language meaning it doesn't do calculations or make decisions. It's a markup language that annotates content so the browser knows how to present it.

Core Concepts at a Glance

🏷
Tags & Elements
Elements are defined with opening and closing tags like <p>...</p>. Some are self-closing like <img/>.
🔗
Attributes
Extra information added inside the tag: href, src, class, id. They give elements meaning and behaviour.
🌳
DOM Tree
The browser turns HTML into a tree of objects (the DOM). JavaScript reads and modifies this tree to make pages dynamic.
♿
Semantic HTML
Tags like <header>, <nav>, <article> describe meaning, helping screen readers and search engines.

Forms & User Input

Forms are how users send data to a server like logins, sign-ups, search bars and more. The <form> element wraps inputs, selects, checkboxes, and a submit button.

Links & Navigation

The <a href=""> (anchor) tag creates hyperlinks which is the fundamental mechanism of the web. Relative links navigate within a site; absolute links point anywhere on the internet.

<!-- Page structure --> <!DOCTYPE html> <html lang="en"> <head> <title>My Page</title> </head> <body> <header> <nav><a href="/">Home</a></nav> </header> <main> <h1>Hello World!</h1> <p>A paragraph of text.</p> <img src="photo.jpg" alt="A photo"/> </main> </body> </html>

📋 Quick-Reference: Common HTML Tags

  • <h1>–<h6> — Headings, h1 is most important
  • <p> — Paragraph of text
  • <a href=""> — Hyperlink
  • <img src="" alt=""> — Image
  • <ul> / <ol> — Unordered / Ordered list
  • <div> / <span> — Generic block / inline container
  • <form>, <input>, <button> — User input
  • <header>, <nav>, <main>, <footer> — Semantic layout
  • <table>, <tr>, <td> — Tabular data
  • <video>, <audio> — Media embeds
