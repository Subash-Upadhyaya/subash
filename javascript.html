What is JavaScript?

JavaScript (JS) is a full programming language which unlike HTML and CSS, can make decisions, perform calculations, loop through data, and communicate with servers.

JavaScript (JS) is a full programming language which unlike HTML and CSS, can make decisions, perform calculations, loop through data, and communicate with servers.

It runs in the browser to control the page, and also on the server via Node.js. One language for the entire web stack.

Core Concepts at a Glance

📦
Variables & Types
let and const store values. Types include strings, numbers, booleans, arrays, and objects.
🔁
Functions & Loops
Reusable code blocks. for, while, and .forEach() repeat tasks over data.
🌳
DOM Manipulation
Select any element with querySelector(), change its text, style, or classes. Add or remove elements on the fly.
🖱
Events
Respond to user actions: click, input, keydown, submit. Attach them with addEventListener().
📋
Arrays & Objects
Arrays store lists; objects store key-value pairs. Most data in real apps is one of these two structures.
⏳
Async & Fetch
fetch() loads data from APIs. async/await lets you handle it without messy callbacks.

The Fetch API — Talking to Servers

Fetch is how JavaScript requests data from a back-end API without reloading the page. This is the core of modern web apps.

// Variables, functions, conditionals const scores = [85, 92, 78, 96]; const avg = scores.reduce((a, b) => a + b) / scores.length; // DOM manipulation const el = document.querySelector('#result'); el.textContent = `Average: ${avg}`; el.classList.add('highlight'); // Event listener document.querySelector('button') .addEventListener('click', () => { el.style.color = 'gold'; }); // Fetch data from an API async function loadUser() { const res = await fetch('/api/user'); const data = await res.json(); console.log(data.name); }
