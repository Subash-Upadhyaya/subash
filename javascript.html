JavaScript — The Language of the Web

JavaScript makes webpages interactive. Things can actually happen when a user clicks a button, fills in a form, or interacts with the page.

What is JavaScript?

JavaScript is the programming language of the web. It makes webpages interactive so things can actually happen when a user clicks a button or fills in a form.

Unlike HTML and CSS which are declarative (they describe structure and appearance), JavaScript is a full programming language — it can make decisions, loop through data, respond to user actions, and communicate with servers.

Core JavaScript Concepts

  • Variables — Store values using let, const
  • Functions — Reusable blocks of code
  • Events — React to user actions like click, input
  • DOM Manipulation — Change HTML/CSS from JavaScript
  • Conditionals — if/else logic and decision making
  • Arrays & Objects — Organise and work with data

DOM Manipulation

The DOM (Document Object Model) is the browser's representation of the HTML page as a tree of objects. JavaScript can select, modify, create, and delete any element on the page.

// Generate a random number const min = 1; const max = 100; const random = Math.floor( Math.random() * (max - min + 1) + min ); // Update the DOM const display = document.getElementById('result'); display.textContent = random; // Event listener const btn = document.querySelector('button'); btn.addEventListener('click', () => { // code runs when button clicked });

JavaScript in the Stack

  • Front-end — Runs in the browser, handles UI events
  • Back-end (Node.js) — Runs on a server, handles APIs
  • Frameworks — React, Vue, and Angular build on JS
  • MEAN/MERN — JavaScript used throughout the whole stack
