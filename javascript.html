JavaScript

JavaScript is what actually makes pages do things. It's the only programming language that runs natively in every browser, and via Node.js it runs on servers too.

What is JavaScript?

Unlike HTML and CSS, JavaScript is a real programming language. It can make decisions, loop through data, respond to what users do, and fetch information from servers — all without reloading the page.

It runs in the browser where it controls the DOM, and also server-side via Node.js. One language, front and back, which is a big part of why so many developers use it for everything.

Core Concepts

📦
Variables & Types
Use let for values that change, const for things that shouldn't. Types — string, number, boolean, array, object — are inferred automatically.
🔁
Functions & Loops
Reusable blocks. for, while, and array methods like .forEach() handle repetition. Arrow functions (() => {}) are a shorter way to write them.
🌳
DOM Manipulation
Grab any element with querySelector(), then change its text, style, or classes. You can also create new elements or delete existing ones.
🖱
Events
Respond to what users do: click, input, keydown, scroll. Attach handlers with addEventListener().
📋
Arrays & Objects
Arrays hold ordered lists. Objects hold key-value pairs. Almost every API response you'll ever deal with is one of these two shapes.
⏳
Async & Fetch
fetch() loads data from a URL without refreshing the page. async/await makes the code read like normal top-to-bottom logic instead of nested callbacks.

Code Examples

JavaScript // scores array — reduce adds them all up, then divide by length for average const scores = [85, 92, 78, 96]; const average = scores.reduce((acc, n) => acc + n, 0) / scores.length; // select element, update text, add class const box = document.querySelector('#result'); box.textContent = `Average score: ${average}`; box.classList.add('highlight'); // click listener — swap colour on click document.querySelector('button') .addEventListener('click', () => { box.style.color = '#f5c842'; }); // async/await — wait for the fetch, then wait for the JSON parse async function getUser() { const resp = await fetch('/api/user'); const data = await resp.json(); console.log(data.name); }
