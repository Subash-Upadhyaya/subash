Bootstrap Framework

Bootstrap is the most widely used CSS framework around. It gives you a 12-column grid, ready-made components, and responsive utilities so you're not writing the same navbar and button code from scratch every time.

What is Bootstrap?

Bootstrap is a free, open-source CSS framework — originally built at Twitter, now independent. The idea is to solve the boring, repetitive parts of web UI so you can focus on the actual design decisions.

Everything is built around a 12-column grid and a set of utility classes that map directly to CSS properties. Knowing Bootstrap is genuinely useful because it's everywhere — most company codebases, templates, and dashboards use some version of it.

Core Concepts

⬜
12-Column Grid
The row width is divided into 12 equal slots. Elements claim however many they need: col-6 is half, col-4 is a third, col-3 is a quarter.
📱
Responsive Breakpoints
Stacking classes changes layout per screen size: col-12 col-md-6 means full width on mobile, half width on tablets and above.
🧱
Components
Navbar, Card, Modal, Alert, Badge, Carousel, Accordion — pre-built UI blocks that just work out of the box. Add the class names, done.
🛠
Utility Classes
Single-purpose helpers: mt-3 (margin-top), d-flex, text-center, fw-bold. You compose your layout from these instead of writing custom CSS for everything.

Grid Syntax

HTML + Bootstrap <!-- structure is always: Container → Row → Columns --> <div class="container"> <div class="row"> <!-- stacks on mobile (col-12), splits 50/50 on md and above --> <div class="col-12 col-md-6">Left</div> <div class="col-12 col-md-6">Right</div> <!-- equal thirds on lg — good for feature/service cards --> <div class="col-lg-4">Card 1</div> <div class="col-lg-4">Card 2</div> <div class="col-lg-4">Card 3</div> </div> </div> <!-- utility classes for spacing and flex layout --> <div class="d-flex justify-content-center gap-3 mt-4"> <button class="btn btn-primary">Primary</button> <button class="btn btn-outline-secondary">Outline</button> </div>

Why Use It?

Reasons Bootstrap is worth knowing

  • Responsive layouts take minutes instead of hours — the grid does the heavy lifting
  • Consistent cross-browser behaviour without writing browser-specific fixes
  • Huge ecosystem: themes, templates, and third-party component libraries
  • Prototyping is much faster — great when you want to test a layout idea quickly
  • The docs have live examples for almost everything, very easy to learn from

class="col-12" — uses all 12 columns (full width)

How it works: The row divides its width into 12 equal units. Each column spans some of those units. Numbers in a row should add up to 12 for a clean layout — going over causes wrapping.
