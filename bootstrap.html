Bootstrap Framework

A free, open-source CSS framework that gives you ready-made components and a powerful 12-column grid system — so you don't have to write all the CSS yourself.

What is Bootstrap?

Bootstrap is a free CSS framework that gives you ready-made components and a grid system so you do not have to write all the CSS yourself. It was created by Twitter and is now one of the most popular front-end frameworks in the world.

I found it really helpful because it saved a lot of time — instead of writing custom CSS for buttons, cards, and navbars from scratch, Bootstrap provides these out of the box with professional-looking defaults.

Why Use Bootstrap?

  • Speed — Pre-built components let you prototype and ship faster
  • Consistency — Uniform styling across all browsers
  • Responsive by default — Mobile-first design philosophy
  • Community — Huge ecosystem of themes, plugins and resources

The 12-Column Grid

The grid system works by dividing the page into 12 columns. You can specify how many columns each element should take up using classes like col-6 (half) or col-4 (one third). Columns are wrapped in a row.

<!-- Bootstrap Grid Example --> <div class="container"> <div class="row"> <!-- Takes 4 of 12 cols = 1/3 width --> <div class="col-4">Left</div> <!-- Takes 8 of 12 cols = 2/3 width --> <div class="col-8">Right</div> </div> </div>

Bootstrap Responsive Breakpoints

  • col- — Extra small (<576px)
  • col-sm- — Small (≥576px)
  • col-md- — Medium (≥768px)
  • col-lg- — Large (≥992px)
  • col-xl- — Extra large (≥1200px)
