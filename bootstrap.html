Bootstrap Framework

A free, open-source CSS framework with a grid system, pre-built components, and utility classes by the help of which we can build professional, responsive sites faster.

What is Bootstrap?

Bootstrap is the world's most popular CSS framework, originally created by Twitter. It gives you a ready-made set of styles, components, and a powerful grid system so you don't need to write everything from scratch.

It's mobile-first by default as the styles apply to small screens first, and you use breakpoints to adjust for larger ones.

What Bootstrap Gives You

📐
12-Column Grid
Divide any row into up to 12 columns. Use col-6 for half width, col-4 for a third. Columns flex automatically.
🧩
Components
Navbars, cards, buttons, modals, badges, alerts, carousels — all pre-styled and ready to drop in.
🛠
Utility Classes
Quick styling without writing CSS: mt-3 (margin-top), text-center, d-flex, fw-bold.
📱
Responsive Breakpoints
Five breakpoints: sm, md, lg, xl, xxl. Mix them: col-12 col-md-6 col-lg-4.
🎨
Colour System
Semantic colours: primary, success, danger, warning. Apply via classes or SASS variables.
✍️
Forms & Validation
Styled form controls, labels, groups, and built-in validation states — all consistent across browsers.

How to Include Bootstrap

<!-- Option 1: CDN link in <head> --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/[email protected]/dist/css/bootstrap.min.css"/> <!-- Option 2: npm install --> npm install bootstrap <!-- Basic grid usage --> <div class="container"> <div class="row"> <div class="col-12 col-md-6 col-lg-4">Card</div> <div class="col-12 col-md-6 col-lg-4">Card</div> <div class="col-12 col-md-12 col-lg-4">Card</div> </div> </div>

Bootstrap vs Writing CSS Yourself

  • Speed — Prototype in hours instead of days
  • Consistency — Uniform look across all browsers
  • Community — Massive ecosystem of themes and plugins
  • Trade-off — Can look "generic" if not customised
